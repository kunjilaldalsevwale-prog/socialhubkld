💾 Backup & Restore
📤 Export all data
Downloads all your posts, ideas, calendar entries, settings and team data as a JSON file. Use this to backup your data or move it to another device/browser.
📥 Import / Restore data
Restore from a previously exported backup file. This will replace all current data with the backup.

Use this to migrate your local data to Vercel:
1. Open your local index.html file → Settings → Export
2. Open your Vercel URL → Settings → Import → pick the file
⚠️ Why is my data missing on Vercel?
Your local file and Vercel are separate — data added locally doesn't automatically appear online. Use Export → Import to move your data to Vercel once, then set up JSONBin sync so all future changes are shared automatically across all devices and team members.
